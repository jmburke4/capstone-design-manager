Capstone Design Manager

A tool for organizing and managing capstone design projects

About the Project

Capstone Design Manager is a project focused on simplifying how capstone design courses handle teams, milestones, documentation, and reviews. The goal is to provide a clean, intuitive experience for students and faculty.

Features

  • Team and project organization
  • Milestone tracking
  • Document and deliverable management
  • Advisor and instructor oversight
