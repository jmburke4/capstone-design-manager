Capstone Design Manager

A platform for organizing and managing senior design projects

About the Project

This Computer Science Capstone project is sponsored by Dr. Monica Anderson Herzog. The purpose of this project is to create a platform for Dr. Anderson to manage new and ongoing senior design projects. This platform should allow project sponsors (who may or may not be affiliated with the University of Alabama) to submit and update projects for Dr. Anderson to approve of and assign to students each semester. This platform should allow students to submit their preferences for which project they wish to work on, so that this may be considered by Dr. Anderson before matching students to projects each semester. This project is initially being designed for the Computer Science and Cybersecurity programs, but may be expanded to support other programs within the college of engineering at a later semester.

See the project deliverables →

Features

  • Submission of projects by sponsors
  • Recording of student project preferences
  • Matching students to projects
  • Instructor administration and oversight
